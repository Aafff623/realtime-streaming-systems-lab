# Languages and runtimes

## 后端

- Java 17
- Maven 多模块
- XML、YAML、SQL
- Spring Boot 3.0.4
- Dubbo 3.2.0-beta.3

## 前端

- JavaScript ES5/ES6
- Vue 2.6.10
- Element UI 2.x
- HTML5、CSS3

## 文档与工具

- Markdown
- PowerShell 或 Python 静态 HTTP 服务
- DOCX/PDF 作为上游只读资料

## 命名约定

- 新文档和目录使用 kebab-case。
- Java 代码沿用现有 `org.qiyu.live` 包结构。
- 新增前端代码在现代化改造前保持现有页面边界，不混入未经批准的框架迁移。
