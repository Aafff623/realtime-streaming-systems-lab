# Assets

## 目录

- `backup/`: 上游 ZIP 与 DOCX 原始媒体，本地只读备份，不提交。
- `images/readme/`: README 配图契约目录。
- `images/avatar/`: 从 Web 上游整理的样例头像。
- `images/icon/`: 从 Web 上游整理的 Logo、状态和支付样例图标。
- `images/knowledge/`: 从上游讲义筛选出的安全知识配图。
- `video/`: 演示视频。
- `theme/ppt/`: 演示文稿。
- `theme/script/`: 项目介绍稿和逐字稿。

## 规则

- 不使用 `docs/images/`。
- 原始包不覆盖、不重打包。
- 第三方截图进入可公开目录前必须脱敏。
- README 图片文件名以 `assets/images/readme/README.md` 为准。
- 图片来源、尺寸和 SHA-256 记录在 `images/manifest.json`。
