# Triage labels

## Category roles

| Canonical role | 本地标签 |
|---|---|
| `bug` | `bug` |
| `enhancement` | `enhancement` |

## State roles

| Canonical role | 本地标签 |
|---|---|
| `needs-triage` | `needs-triage` |
| `needs-info` | `needs-info` |
| `ready-for-agent` | `ready-for-agent` |
| `ready-for-human` | `ready-for-human` |
| `wontfix` | `wontfix` |

每个已分诊 Issue 必须只有一个 Category 和一个 State。状态冲突时停止处理并交由用户决定。
