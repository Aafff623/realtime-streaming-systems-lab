# Upstream docs context

## 内容

`upstream-docs/` 保存两组课程资料：

- 根目录: 42 篇 DOCX 课程讲义和空 README。
- `reference-pdf/`: 49 份 PDF 与 1 份 DOCX 参考资料，原文件名存在编码损坏。

## 使用原则

- 目录只读，不直接修改原稿。
- 原稿用于理解课程设计意图，不能覆盖源码事实。
- 文档中的 Java、SQL、YAML 和 Docker 片段需要核对代码后再转入 `docs/knowledge/`。
- 第三方控制台截图和账号信息必须脱敏。

## 知识迁移优先级

1. Docker、中间件和 Maven 环境。
2. Nacos 与 ShardingSphere。
3. Gateway 鉴权、登录和短信。
4. Netty IM 与 Router。
5. 直播 PK、红包和带货。
